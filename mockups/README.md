# Mobile Mockups

Wenn sich die SVG-Dateien in deiner IDE nicht gut öffnen lassen, nutze diese HTML-Dateien:

- `open-this-first.html`
- `mobile-screens.html`

`mobile-screens.html` enthaelt sechs besser lesbare mobile Screens:

- Dashboard
- Tagesansicht
- Aufgabe erstellen
- Wiederholungen
- Statistik
- Notizen

Zusätzlich liegen weiterhin die ersten SVG-Mockups im Ordner:

- `day-view.svg`
- `week-view.svg`
- `stats-view.svg`
- `notes-view.svg`

Empfehlung:
Oeffne `mockups/mobile-screens.html` direkt im Browser.
